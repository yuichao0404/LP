# 🎨 スライド2：セクション1実践 - 画像1作成（Canva編集）

## 🎯 セクション1の画像1を作成します

### 📋 準備するもの

- [ ] Canvaアカウント（無料版でもOK）
- [ ] 商品画像（練習用：ダウンロードリンクはマニュアル参照）
- [ ] 作業時間：10-15分

---

## 🔧 ステップ・バイ・ステップ実践

### ステップ1：Canvaでキャンバス作成（1分）

```
1. Canvaを開く
2. 「デザインを作成」をクリック
3. カスタムサイズを選択
   - 幅：1920px
   - 高さ：1080px
4. 「新しいデザインを作成」をクリック
```

✅ **チェックポイント**: 白いキャンバスが表示されましたか？

---

### ステップ2：背景色を設定（2分）

**温活zenは健康食品カテゴリ → ミントグリーン＋ベージュ**

```
1. 左メニューから「素材」をクリック
2. 「グラデーション」で検索
3. 緑系のグラデーションを選択
4. またはカスタム背景色を設定：
   - 色1：#B8E6D5（ミントグリーン）
   - 色2：#F5F1E8（ベージュ）
```

**💡 プロのコツ**: グラデーションは左上→右下に流すと自然

✅ **チェックポイント**: 背景が落ち着いた緑系になっていますか？

---

### ステップ3：商品画像をアップロード（3分）

```
1. 左メニュー「アップロード」をクリック
2. 「ファイルをアップロード」で商品画像を選択
3. アップロードされた画像をキャンバスにドラッグ
4. 画像を選択 → 「背景を削除」をクリック（Pro版）
   ※無料版の場合：消しゴムツールで手動削除
```

**⚠️ 注意**: 商品の背景が透明になるまで丁寧に削除しましょう

✅ **チェックポイント**: 商品の背景が透明になりましたか？

---

### ステップ4：商品画像を配置（2分）

**パターン1（説明型）は「中央配置・大きめ」**

```
1. 商品画像を選択
2. キャンバスの中央に配置
3. サイズ調整：キャンバスの50-60%の大きさ
4. 位置：上下に余白を確保（テキスト用）
```

**配置の目安**:
```
┌──────────────────────┐
│                      │ ← 上部余白（テキスト用）
│                      │
│      [商品画像]       │ ← 中央に大きく配置
│                      │
│                      │ ← 下部余白（テキスト用）
└──────────────────────┘
```

✅ **チェックポイント**: 商品が中央に配置され、上下に余白がありますか？

---

### ステップ5：テキストオーバーレイを追加（4分）

**追加するテキスト内容**:
```
毎朝たった1分で
健康習慣が身につく
梅白湯メーカー
```

**フォント設定（パターン1用）**:
- フォント：Noto Sans JP
- 太さ：Bold
- サイズ：48-60pt
- 色：#000000（黒）
- 配置：上部または下部（商品に重ならない位置）

```
1. 左メニュー「テキスト」をクリック
2. 「見出しを追加」を選択
3. テキストを入力
4. フォントを「Noto Sans JP」に変更
5. 太さを「Bold」に変更
6. サイズを50ptに設定
7. 色を黒に変更
8. 商品の上部または下部に配置
```

**💡 プロのコツ**: テキストに「ドロップシャドウ」を追加すると読みやすい

✅ **チェックポイント**: テキストが商品に重なっていませんか？

---

### ステップ6：最終調整とダウンロード（3分）

**最終チェック**:
```
□ サイズは1920×1080pxになっているか
□ 商品画像の背景は適切に処理されているか
□ テキストが読みやすいか
□ 商品にテキストが重なっていないか
□ 全体のバランスは良いか
```

**ダウンロード**:
```
1. 右上「共有」→「ダウンロード」
2. ファイルの種類：PNG
3. ファイル名：温活zen_画像1_説明型.png
4. 「ダウンロード」をクリック
```

✅ **完了！** 画像1が完成しました！
